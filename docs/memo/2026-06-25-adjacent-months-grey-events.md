# 2026-06-25 adjacent-months-grey-events

## 実施内容

- カレンダー表示で、前月の月末および翌月の月初の日付セルに対しても、登録されている予定を表示するように変更しました。
- 前月・当月・翌月の合計3ヶ月分の予定データを並列で取得するように `components/CalendarApp.tsx` を修正しました。
- 非対象月（前月・翌月）の予定カード（`SeminarCard`）は、クリック不可・ホバーアニメーション無しの静的な `div` 要素（グレー表示・半透明）として描画されるようにしました。
- モバイル表示用のコンパクトな予定チップも、非対象月はグレーアウト表示になるよう調整しました。
- 非対象月のカレンダー日付セル（`div`）自体の `cursor-pointer` やクリック・アクティブ色変更などのスタイルを排除し、`cursor-default` に変更して、誤クリックなどの混乱を防ぐように改善しました。

## レビュー観点

- 開発サーバーおよびビルド検証（`npm run check:dev`、`npm run check`）が通ること。
- 非対象月の予定カードをクリックした際に、編集・詳細モーダルが開かないこと。
- 非対象月の予定カードにホバーしても、スケールアニメーションやホバーシャドウが発生しないこと。
- 当月の予定カードは通常通りカラーで表示され、ホバーやクリックが正常に動作すること。
