# 2026-06-25 logo-style-adjustment

## 実施内容

- ヘッダー左端のロゴ表示を調整し、より直接的かつ視覚的に映えるデザインに改良しました：
  - **外枠（ボーダー）と背景色の廃止**: 従来ロゴを囲んでいた白いカード枠（`border`、`shadow-sm`、`bg-white`、`rounded-lg`）を排除し、ロゴ画像を直接ヘッダーに配置するように変更しました。
  - **アイコンサイズの拡大**: 表示面積を少し大きくし、モバイルで `w-10 h-10`（40px）、デスクトップ/タブレット表示で `w-11 h-11`（44px）に拡張しました。これにより、ヘッダータイトル「班ゼミカレンダー」とのバランスが向上し、視認性が高まりました。
  - **アスペクト比の維持**: `object-contain shrink-0` を適用し、アイコン画像が歪むことなくクリアに表示されるようにしました。

## レビュー観点

- ヘッダーのロゴ画像に余分な枠線や影が表示されず、直接配置されていること。
- 文字サイズやヘッダーの高さに対して適切なアイコンサイズ（モバイル：40px、デスクトップ：44px）で表示されていること。
- `npm run check:dev` がエラーなくパスすること。
