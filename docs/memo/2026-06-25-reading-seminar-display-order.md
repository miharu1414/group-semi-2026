# 2026-06-25 reading-seminar-display-order

## 実施内容

- 同じ日付に複数予定がある場合、輪読ゼミを最後に表示するソート処理を追加した。
- `lib/seminar-sort.ts` に予定表示順の比較関数を切り出した。
- `CalendarApp` と `CalendarView` の並び順を共通比較関数に揃えた。

## 背景

輪読ゼミが上に表示されると、同じ日の全体ゼミや研究共有などがカレンダー上で見えにくくなるため。

## 表示ルール

1. 同じ日付では、輪読ゼミ以外を先に表示する。
2. 輪読ゼミは同日の最後に表示する。
3. 同じ優先度内では、開始時刻、タイトル順に表示する。
