# 2026-06-25 premium-ui-and-logo-update

## 実施内容

- ヘッダー左端のロゴマークを、アップロードされた教授と電球のイラスト（`public/logo.png`）に変更しました。
- アプリ全体の配色が白く淡白すぎる点を改善するため、以下のデザイン改良を行いました：
  - アプリの背景に、薄いインディゴとティールの上品なグラデーションのぼかし（Radial spot gradient）を配置し、視覚的な奥行きを持たせました。
  - ヘッダーを半透明のガラスモーフィズム（`bg-white/80 backdrop-blur-md`）に変更し、ぼかし背景が奥で透けるようにしました。
  - ヘッダーの「班ゼミカレンダー」の文字に鮮やかなグラデーション（`indigo` ➔ `violet`）を適用し、タイトルのブランド表現を向上させました。
  - カレンダー上部のナビゲーションバーを半透明にし、曜日のヘッダー背景や日付セルを少し透過させ、背景グラデーションと馴染むように調和させました。
  - カレンダーの日付セルのホバー時・アクティブ時の変化をより滑らかなトランジションに調整しました。
  - 下部のメモ・連絡事項（`NoticeBoard`）のパネルを半透明ガラス調にし、メモカード自体も微小な影とホバーエフェクト（`shadow-xs` ➔ `hover:shadow-sm`）を持たせることで、立体感とプレミアム感を演出しました。
